# 11-11 — Gestión y respuesta a alertas

```text
Objetivo:
Definir, ajustar y responder a alertas operativas.

Inputs requeridos:
- alerta: [NOMBRE / DESCRIPCIÓN]
- umbral: [VALOR]
- responsable: [NOMBRE]
- acción automática: [SÍ / NO]

Flujo:
1. Definir alerta y umbral.
2. Configurar notificación y canal.
3. Documentar acción automática (si aplica).
4. Procedimiento de respuesta:
   - Diagnóstico inicial.
   - Acciones correctivas.
   - Escalamiento si no se resuelve en [X] minutos.
5. Registro de incidentes asociados.

Campos personalizables:
- [ALERTA], [UMBRAL], [RESPONSABLE], [ACCIÓN]
```

## Descripción

Prompt para gestión integral de alertas: definición, ajuste, respuesta y documentación. Permite personalizar umbrales y responsables.

---

Ejemplo de uso:

- alerta: CPU alta
- umbral: 90%
- responsable: Mario Torres
- acción automática: SÍ
